# vfox-frankenphp

[![CI](https://github.com/jbox-web/vfox-frankenphp/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-frankenphp/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [frankenphp](https://github.com/php/frankenphp)

## Install

```
mise plugin install frankenphp https://github.com/jbox-web/vfox-frankenphp.git
```
